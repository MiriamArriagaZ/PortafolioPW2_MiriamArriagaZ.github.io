Definición de objetivos

Objetivos Generales:

  • Crear un sitio web dinámico y fácil de entender donde se facilite la compre y venta de libros de forma versátil y accesible

Objetivos Específicos:

  • Conseguir que el usuario está cómodo con la interfaz en todo momento.
  • Facilitar la adquisición de libros.
  • Promover la lectura.
  • Diseñar un boceto de la nueva página web.
  • El diseño web será amigable con el usuario.
  • El sitio web presentará optimización adecuada en ventas y eventos.
  • El sitio web será actualizado constantemente.
